# UI 原型设计

## 1. 主窗口布局

```
┌──────────────────────────────────────────────────────────────┐
│  网络设备采集分析平台  v1.0                    ─ □ ×      │
├──────────┬───────────────────────────────────────────────────┤
│          │                                                    │
│  ⚙ 配置  │   [页面标题]                        [操作按钮区]   │
│  ├ 📡连接 │                                                    │
│  ├ 🖥️设备 │   ┌─ 内容区域 ──────────────────────────────┐   │
│  ├ 📐场景 │   │                                             │   │
│  ├ ⌨️命令 │   │   当前页的具体内容                           │   │
│  │        │   │   (不同页面切换显示不同内容)                  │   │
│  📋 任务  │   │                                             │   │
│  ├ ➕新建  │   └─────────────────────────────────────────────┘   │
│  ├ 📊历史  │                                                    │
│  └ ⏰定时  │                                                    │
│          │                                                    │
├──────────┴───────────────────────────────────────────────────┤
│  就绪      │ 上次任务: 路由放通检查  17:30 ✓  │  3个冲突     │
└──────────────────────────────────────────────────────────────┘
```

左侧导航：配置区 + 任务区，选中高亮
内容区：根据导航切换
状态栏：左下就绪状态，右下最近任务摘要

│                                                                      │
│ ┌─ 连接列表 ─────────────────────────────────────────────────────┐  │
│ │ 名称          │ 主机              │ 端口 │ 用户    │ 状态      │  │
│ ├─────────────────────────────────────────────────────────────────┤  │
│ │ 大云堡垒机    │ bastion.dayun.com │ 22   │ admin  │ ✓ 可用    │  │
│ │ 苏州堡垒机    │ suzhou-bastion.c  │ 22   │ admin  │ ✓ 可用    │  │
│ │ 测试连接      │ 10.0.0.1         │ 22   │ root   │ ✗ 不可达  │  │
│ └─────────────────────────────────────────────────────────────────┘  │
│                                                                      │
│ ┌── 编辑/新增连接 ─────────────────────────────────────────────────┐ │
│ │  名称: [大云堡垒机              ]                                 │ │

│ │  私钥路径: [C:\Users\xxx\.ssh\id_rsa  ] [浏览...]                │ │
│ │  私钥密码: [********             ]                                │ │
│ │                                                                   │ │
│ │  Expect流程:                                                      │ │
│ │  ┌──────────────────────────────────────────────┐                │ │
│ │  │ #  看到                   发送                │                │ │
│ │  │ 1  Opt or ID>:            n                  │                │ │

│ │  │ 3  Opt or ID>:            ={device_ip}       │                │ │

Expect 流程可动态增删行。`{device_ip}` 用文本标签显示，运行时自动替换。
点击"测试连接" → 后台 SSH 尝试连接 → 弹窗显示成功/失败详情。
